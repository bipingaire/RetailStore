# .env.production

# ⚠️ IMPORTANT: DO NOT COMMIT THIS FILE TO GIT!

This file contains production credentials including:
- OpenAI API key
- Google Search API key  
- Database password
- JWT secrets

**Security Notes:**
- `.env.production` is in `.gitignore`
- Only keep this file on your production server
- Use environment-specific secrets management in production

**For deployment:**
1. Copy `.env.production` to your server
2. Update passwords and secrets
3. Never commit actual credentials to Git

**For team members:**
- Share credentials securely (1Password, etc.)
- Each developer should have their own `.env.local`
- Production credentials should be separate
